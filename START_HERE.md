## 🎬 Quick Overview

This MCP server gives AI assistants access to your Google Workspace:

- 📧 Gmail - Read, send, manage emails
- 📁 Drive - Access, create, share files
- 📅 Calendar - View, create, manage events
- 📝 Docs - Read, edit documents
- 📊 Sheets - Manage spreadsheets
- And 7 more services!

## 🏗️ Deployment Flow

```
1. Get Google OAuth credentials (2 min)
   ↓
2. Create resource in Coolify (1 min)
   ↓
3. Set environment variables (1 min)
   ↓
4. Deploy (1 min)
   ↓
5. Update external URL (30 sec)
   ↓
6. Test & connect clients (30 sec)
```
