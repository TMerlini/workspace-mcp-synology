================================================================================
  DEPLOYMENT OVERVIEW
================================================================================

Step 1: Get Google OAuth Credentials (2 minutes)
   - Go to console.cloud.google.com
   - Create OAuth Desktop credentials
   - Enable required APIs

Step 2: Deploy to Coolify (3 minutes)
   - Create new resource in Coolify
   - Point to this repository
   - Set environment variables
   - Deploy

Step 3: Configure External URL (30 seconds)
   - Copy Coolify-assigned URL
   - Update WORKSPACE_EXTERNAL_URL
   - Redeploy

Step 4: Test & Connect (1 minute)
   - Test health endpoint
   - Connect MCP client
   - Start using!

================================================================================
  KEY CONFIGURATION
================================================================================

Required Environment Variables:
   GOOGLE_OAUTH_CLIENT_ID          (from Google Cloud)
   GOOGLE_OAUTH_CLIENT_SECRET      (from Google Cloud)
   MCP_ENABLE_OAUTH21=true
   WORKSPACE_MCP_STATELESS_MODE=true
   WORKSPACE_EXTERNAL_URL          (from Coolify after deploy)

Recommended Settings:
   TOOL_TIER=core                  (start with essential tools)
   WORKSPACE_MCP_PORT=8000
