================================================================================
  HOW TO PUSH TO GITHUB
================================================================================

Your GitHub Repository:
https://github.com/TMerlini/Google-Workspace-MCP-Synology

================================================================================
  QUICK INSTRUCTIONS
================================================================================

Since Git is not installed on your system, here are your options:

OPTION 1: Install Git (Recommended)
------------------------------------
1. Download Git: https://git-scm.com/download/win
2. Install with default settings
3. Open PowerShell in Z:\Googlemcp
4. Run: .\push-to-github.ps1
5. Follow the prompts

OPTION 2: Use GitHub Desktop (Easiest)
---------------------------------------
1. Download: https://desktop.github.com/
2. Install and sign in to GitHub
3. File → Add Local Repository → Choose Z:\Googlemcp
4. Click "Publish repository"
5. Done!

OPTION 3: Manual Upload via Web
--------------------------------
1. Compress Z:\Googlemcp to a ZIP file
2. Go to: https://github.com/TMerlini/Google-Workspace-MCP-Synology
3. Click "uploading an existing file"
4. Upload the ZIP
5. Commit changes
